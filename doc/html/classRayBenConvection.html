Detailed Description

RayBenConvection is the core of the numerical simulation.

This class performs perturbative calculations on a fixed environment, describing fluid dynamics evolution according to Navier-Stokes equations.

This function resolves Partial Pivoting calculations with LU decomposition.

Calling this function will return an std::vector of Eigen Matrices consisting of the two matrices L and U obtained during the decomposition.

See also
Eigen Partial Pivoting (LU decomposition)

The input matrix A is decomposed as A = PLU where L is unit-lower-triangular, U is upper-triangular, and P is a permutation matrix.

Prandtl number.

Dimensioneless number defined as the ratio of momentum diffusivity to thermal diffusivity. For Pr < < 1 the thermal diffusivity dominatest the behavior, while for Pr > > 1 the momentum diffusivity dominates.

◆ m_Ra

double RayBenConvection::m_Ra
private

Rayleigh number.

Dimensionless number associated with buoyancy-driven flow, also known as natural convection. It characterises the fluid's flow regime: below a critical value, denotes laminar flow, and heat transfer is by conduction rather than convection. Over the critical value, it denotes turbulent flow with convection cells.
